Installed Requirements
Requirement 1:
- Install all the font files found in the "ForzaFont" folder (preferred)
Requirement 2:
- Download & Install Simhub
- Go through the initial first launch process and set-up SimHub as you wish (units, colors, games etc)
- Configure the UDP and Telemetry settings for Forza Motorsport 2023
Requirement 3:
- In Forza's Gameplay & Hud Settings disable/turn OFF the following:
- Lap Time, Position, Score, Pit Stops
- Speedometer
- Car Progression
- Drift HUD
- Quick Action
- Track Mastery
- Race Objectives
- Rewind Notification
Optional ones:
- Track Caution Messages
- Race Activity Feed
- Race Regulation Messages
Now onto the Overlays
Semi-Automatic Method (shorter, set it and forget it)
NOTE: These will come preloaded with the Base Lap-History-V1 styling and the Automated Tachometer.
How to Change
Step 1:
- Import *ALL* the individual Overlays from the "Overlay-Modules" folder, and for all of them unckeck the "Install Dash Fonts" (as we already installed them).
- Open the folder "Olayouts"
- Copy the desired resolution of the "Motorsport-Ultimate.olayout" and/or "Motorsport-Ultimate-Extended.olayout" file/s
- Find the install location of your SimHub app. Usually C:\Program Files (x86)\SimHub, unless you changed the default install location
- Open the folder "OverlayLayouts", C:\Program Files (x86)\SimHub\OverlayLayouts
- Paste the file/s into there.
Step 1a (To change the Lap-History style):
- Close the SimHub app.
- Open the .olayout file/s with any text editor.
- Search/Look for "Lap-History".
- You'll find 2 entries within the same line for each file.
"DashboardName": "DashTemplates\\Art-Lap-History-Ext-v1\\Art-Lap-History-Ext-v1.djson",
- Change both of the version names.
This
"DashboardName": "DashTemplates\\Art-Lap-History-V1\\Art-Lap-History-V1.djson",
Into this
"DashboardName": "DashTemplates\\Art-Lap-History-V1.1\\Art-Lap-History-V1.1.djson",
Or Into this
"DashboardName": "DashTemplates\\Art-Lap-History-V1.2\\Art-Lap-History-V1.2.djson",
- Save the file/s.
Step 1b (To swap the Tachometer):
- Close the SimHub app.
- Open the .olayout file/s with any text editor.
- Search/Look for "Tachometer".
- You'll find 2 entries within the same line for each file.
"DashboardName":"DashTemplates\\Art-Tachometer-Automated\\Art-Tachometer-Automated.djson",
- Change both of the "Automated".
This
"DashboardName":"DashTemplates\\Art-Tachometer-Automated\\Art-Tachometer-Automated.djson",
Into this
"DashboardName":"DashTemplates\\Art-Tachometer-Manual\\Art-Tachometer-Manual.djson",
- Save the file/s.
Step 2:
- If your Default Overlay Renderer is HTML, we'll need to change the Renderer for the Tachometer.
- Load your Overlay.
- Click "Edit Layout"
- Now you'll need to change the Renderer set for the Tachometer.
- Click on the Gear Icon for the Tachometer.
- Set the Renderer for it to "WPF Render Engine".
